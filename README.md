## Requirements

## Getting Started

- Write your circom program in the `input` directory under the `circuit.circom` name.

- Build the program

```bash
cargo build --release
```

- Run the compilation

```
cargo run --release
```

The compiled circuit and circuit report can be found in the `./output` directory.
